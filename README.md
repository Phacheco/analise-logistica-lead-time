# analise-logistica-lead-time
Análise de performance de entrega (No Prazo vs Atraso) utilizando Excel e Power Query.
Neste projeto, analizei o comportamento das entregas de uma operação logística para entender ocomportamento de atraso das transportadoras. Utilizei o Power Query para limpar e modelar os dados brutos, transformando datas e status em métricas de tempo (Lead Time).

O que eu fiz:
Organizei o fluxo de trabalho em três etapas para manter a integridade dos dados, comecei com os dados brutos (01_RAW), apliquei todo o tratamento e cálculos de datas no Excel (02_PROCESSED) e gerei as visualizações finais e prints do dashboard (03_REPORTS).

O que os dados revelaram:
O ponto mais interessante da análise foi o mês de Junho. Ao cruzar as médias, percebi que ele foi o "ponto fora da curva" teve a maior demora nas entregas atrasadas e, ao mesmo tempo, uma queda na eficiência das entregas no prazo. Isso indica que algo específico aconteceu na operação naquele mês, exigindo uma investigação mais profunda sobre a causa raiz desse gargalo.
